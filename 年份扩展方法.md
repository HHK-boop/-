# Week 3 年份扩展方法说明

## 1. 扩展范围

- 负责板块：北交所。
- 扩展年份：2024年、2023年。
- 数据来源：北京证券交易所股票列表与相关公告接口，入口为 `https://www.bse.cn/nq/listedcompany.html`。
- 样本数量：2024年23家全量处理，2023年10家等距抽样处理，合计33家。

## 2. 抽样规则

2024年北交所上市样本为23家，本次全部纳入。2023年北交所上市样本为77家，样本较多，按上市日期排序后进行等距抽样，选取10家公司，覆盖年初、年中和年末上市样本。

## 3. 代码迁移方法

第二周2025年流程可以复用主体框架，包括清单字段、PDF准备、pypdf解析、关键词定位、候选JSON抽取和字段校验。第三周主要修改点如下：

- 企业清单脚本改为按 `ipo_year` 筛选2024和2023。
- PDF准备脚本针对北交所PDF链接加入浏览器请求头/PowerShell下载分支，解决普通urllib访问北交所披露文件时的重定向和403问题。
- 日志统一改为 `week3_*_log.csv`，避免覆盖Week2日志。
- 输出路径改为 `outputs/week3_sample_outputs/`。
- 组内互查表按任务书Week3模板生成。

## 4. 运行方式

```bash
python code/run_week3_pipeline.py
```

也可以分步运行 `code/01_build_company_list/` 至 `code/07_validate_outputs/` 下的 Week3 脚本。
